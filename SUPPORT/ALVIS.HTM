Installation

The installation for this plugin (ALVIS.EXE) has been created using Nullsoft Scriptable
Install System (NSIS). Installation extracts the following files into WINAMP's Plugins
directory:

  • 1. VIS_ALVIS.DLL - the visualization DLL or simply the ALVIS Plugin

  • 2. BOOM.BMP - the Boom Box background

  • 2. ALVIS.HTM - this HTML help/reference page

  • 2. ALVIS.CSS - this page's Style sheet

Features and Usage

It can operate in five modes as of now:

  • # Waveform Analyser (aka) Oscilloscope (erstwhile Ding Dong)
            Ride the waves through the left and right channels.


  • # Spectrum Analyser (formerly called Ping Pong)
            A spectral graph never looked more convincing.


  • # VU Analyser (previously VU Meter)
            A unique blend of lines and bars used to bring out a funky music-meter.


  • # The Grind (special thanks to Prasanna)
            Watch the energetic grind wheels as they roll to the beats.


  • # Boom Box - Explosion (formerly called Experimentz)
            Be prepared for an explosive experience.

In the Visualization Window, in addition to the standard WINAMP commands, use the following:

KeyFunction
  YToggle OSD (On-Screen Display) On/Off
  TToggle Boom Box Background On/Off
 WSwitch Boom Box mode: Waveform and Spectrum
