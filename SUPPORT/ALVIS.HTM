Installation

The installation for this plugin has been created using Nullsoft Scriptable Install System (NSIS). Installation extracts the following files into Winamp's Plugins directory:

  • 1. VIS_ALVIS.DLL - the visualization plugin

  • 2. ALLEG42.DLL - Allegro runtime library (required)

  • 3. BOOM.BMP - the Boom Box background

  • 4. ALVIS.HTM - this help page

top

Features and Usage

It can operate in five modes:

  • # Waveform Analyser - oscilloscope display for left/right channels

  • # Spectrum Analyser - spectral frequency graph

  • # VU Analyser - audio level meter with bars and lines

  • # The Grind - beat-driven animated ellipses

  • # Boom Box - Explosion - radial speaker explosion with waveform or spectrum

In the Visualization Window, in addition to the standard WINAMP commands, use the following:

KeyFunction
  YToggle OSD On/Off
  TToggle Boom Box Background On/Off
 WSwitch Boom Box mode (Waveform/Spectrum)

Building from Source

Requires MinGW/GCC and Allegro 4.x. Set EXTDEV to your SDK root, then run:

BUILD.BAT

Output goes to DIST\. See DEVELOP.MD for details.
